# Инструкция по деплою на Cloudflare Pages

## Cloudflare Pages - идеальный выбор! ✅

**Преимущества:**
- ✅ Бесплатный хостинг
- ✅ Надежный и быстрый (CDN по всему миру)
- ✅ Автоматический SSL сертификат
- ✅ Автоматический деплой из GitHub
- ✅ Поддержка кастомных доменов
- ✅ Неограниченная пропускная способность

## Шаги для деплоя:

### 1. Подготовка репозитория
Репозиторий уже готов: `https://github.com/edtechhacker/sho.git`

### 2. Создание аккаунта Cloudflare
1. Перейдите на https://dash.cloudflare.com/sign-up
2. Создайте бесплатный аккаунт
3. Подтвердите email

### 3. Подключение GitHub к Cloudflare Pages
1. Войдите в Cloudflare Dashboard
2. Перейдите в **Pages** → **Create a project**
3. Выберите **Connect to Git**
4. Авторизуйтесь через GitHub
5. Выберите репозиторий `edtechhacker/sho`
6. Нажмите **Begin setup**

### 4. Настройка деплоя
В настройках проекта укажите:
- **Project name**: `sho` (или любое другое)
- **Production branch**: `main`
- **Build command**: (оставьте пустым, т.к. это статический сайт)
- **Build output directory**: `/` (корень проекта)

**Важно:** Так как у вас статический HTML файл в корне, никаких команд сборки не нужно!

### 5. Подключение домена shontech.ru

#### 5.1. Добавление домена в Cloudflare
1. В Cloudflare Dashboard перейдите в **Pages** → ваш проект
2. Откройте вкладку **Custom domains**
3. Нажмите **Set up a custom domain**
4. Введите `shontech.ru`
5. Cloudflare автоматически проверит DNS записи

#### 5.2. Настройка DNS записей
Если домен еще не в Cloudflare:
1. Перейдите в **DNS** в Cloudflare Dashboard
2. Добавьте домен `shontech.ru` (если еще не добавлен)
3. Cloudflare предоставит DNS серверы (например: `alice.ns.cloudflare.com`)

Если домен уже зарегистрирован:
1. Зайдите в панель управления доменом (где регистрировали)
2. Измените DNS серверы на те, что предоставил Cloudflare
3. Подождите 24-48 часов для распространения DNS

#### 5.3. Настройка DNS записей для Pages
Cloudflare Pages автоматически создаст нужные DNS записи:
- **CNAME** запись для `shontech.ru` → `your-project.pages.dev`
- **A** записи для корневого домена (если нужно)

### 6. Проверка деплоя
1. После подключения репозитория, Cloudflare автоматически задеплоит сайт
2. Вы получите временный URL: `your-project.pages.dev`
3. После настройки домена, сайт будет доступен на `shontech.ru`

## Автоматический деплой

После настройки:
- ✅ Каждый push в `main` ветку автоматически задеплоит изменения
- ✅ Деплой занимает ~1-2 минуты
- ✅ Можно настроить preview деплои для pull requests
